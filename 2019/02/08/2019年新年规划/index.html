2019年新年规划

Posted on 2019-02-08

以此文来记录一些2018年所发生的一些事情以及2019的新年规划，所谓的温故而知新。

2018年虽然成功跳槽，但并不是我理想中的地方，在加上一些其他的事情，其实整年内心是很彷徨、焦虑的。

但2018年也是我感觉知识面以及对技术的理解增长最快的一年，果然是痛并快乐着，古人诚不我欺也。

18年做的一些全栈上的项目，比如埋点管理平台，从数据库设计到api设计再到前端页面，让我看到了作为一个全栈工程师所需要的知识面，也让我看到了作为一个合格的工程师需要具备的一些必要条件：比如源码阅读。

总而言之，2018年让我看到了我自己的不足，让我漂浮的心重新回归，还有很多的事情要做，还有很多的东西要去学习，广阔如星海一般。

就让我们用胡适的一句话来宽慰自己： 进一寸有一寸的欢喜。

对于2019年我做了很多思考，18年基本是在迷茫以及困惑焦虑中度过的，也导致了精神状态的不好，脑子中经常纠结一些有的没的，跟之前在大学的状态一样，很差劲。

希望2019年我可以找回自己的状态，18年真的很差。。。，18年是失败的一年，以前的一些债终于还是要还的。

19年我在这里给自己立一些flag:

  1. 跑步及健身。 把这个作为第一项没有任何问题，感觉目前健身可以救我了。希望自己在偷懒的时候可以想起来这一点：并不要觉得累而不去健身，而是因为不去健身而觉得累。
  2. 开火做饭。没啥原因，外卖真的吃恶心了！
  3. 扩大自己的影响力。看源码 -> 发文章(暂定源码系列的文章) -> 建立自己的github项目(看的源码差不多了考虑自己的github项目，偏node方向)。
  4. 新一年彻底掌握的技术栈： react全家桶 + node(egg midway) + graphql生态 + docker + ts + mysql + shell（暂定） + git(git的彻底掌握)

补充分割线：
现在才意识到一些问题，18年状态完全是因为

  1. 自己累了，想休息，没有一种持续性奋斗的状态
  2. meet some bad things, 影响了我的心绪

总结:
如果自己遇到一些内心纠结，无法摆脱的事情，just rest。 不要浪费自己的时间，有时候停下脚步未必是坏事，按照自己的节奏来，do what you want to do!
